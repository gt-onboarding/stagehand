Stagehand（AI 增强浏览器自动化框架）提供全面的日志能力，助你调试自动化工作流、跟踪执行并定位问题。你可以在开发与生产环境中配置日志级别、结构化输出和调试工具。

## 日志配置

<CodeGroup>
```typescript TypeScript
import { Stagehand } from "@browserbasehq/stagehand";

const stagehand = new Stagehand({
  env: "BROWSERBASE", // or "LOCAL"
  verbose: 1, // 0 = errors only, 1 = info, 2 = debug
});
```

```python Python
from stagehand import Stagehand

stagehand = Stagehand(
    env="BROWSERBASE",  # or "LOCAL"
    verbose=1,  # 0 = errors only, 1 = info, 2 = debug
)
```
</CodeGroup>

### 日志详细级别

- **Level 0**：仅错误——适用于生产环境的最少输出
- **Level 1**：信息——包含成功操作与重要事件
- **Level 2**：调试——包含内部操作在内的全面日志

## 结构化日志

### 日志行格式

每条日志记录均包含结构化信息：

<CodeGroup>
```typescript TypeScript
interface LogLine {
  category: 'browser' | 'action' | 'llm' | 'error' | 'stagehand' | 'cache';
  message: string;
  level: 0 | 1 | 2; // error | info | debug
  timestamp: string;
  auxiliary?: {
    executionTime?: { value: string; unit: string };
    sessionId?: string;
    url?: string;
    [key: string]: any;
  };
}
```

### 自定义日志记录器

<CodeGroup>
```typescript TypeScript
class AdvancedLogger {
  private logFile?: string;
  
  constructor(logFile?: string) {
    this.logFile = logFile;
  }
  
  log = (logLine: any) => {
    const timestamp = new Date().toISOString();
    const colors = {
      browser: '\x1b[34m', // 蓝色
      action: '\x1b[32m',  // 绿色
      llm: '\x1b[35m',     // 洋红
      error: '\x1b[31m',   // 红色
      stagehand: '\x1b[36m', // 青色
      cache: '\x1b[33m',   // 黄色
    };
    
    const color = colors[logLine.category] || '\x1b[0m';
    const reset = '\x1b[0m';
    
    // 控制台彩色输出
    console.log(`${color}[${logLine.category}]${reset} ${logLine.message}`);
    
    // 若有，记录执行时间
    if (logLine.auxiliary?.executionTime) {
      console.log(` ${logLine.auxiliary.executionTime.value}${logLine.auxiliary.executionTime.unit}`);
    }
    
    // 记录附加上下文
    if (logLine.auxiliary && Object.keys(logLine.auxiliary).length > 0) {
      console.log('  Context:', JSON.stringify(logLine.auxiliary, null, 2));
    }
    
    // 文件日志（可选）
    if (this.logFile) {
      const logEntry = {
        timestamp,
        ...logLine
      };
      require('fs').appendFileSync(this.logFile, JSON.stringify(logEntry) + '\n');
    }
  }
}

// 用法
const logger = new AdvancedLogger('./automation.log');
const stagehand = new Stagehand({
  env: "BROWSERBASE",
  verbose: 2,
  logger: logger.log
});
```

```python Python
import json
import os
from datetime import datetime
from typing import Dict, Any, Optional

class AdvancedLogger:
    def __init__(self, log_file: Optional[str] = None):
        self.log_file = log_file
    
    def log(self, log_line: Dict[str, Any]):
        timestamp = datetime.now().isoformat()
        colors = {
            'browser': '\033[34m',   # 蓝色
            'action': '\033[32m',    # 绿色
            'llm': '\033[35m',       # 洋红
            'error': '\033[31m',     # 红色
            'stagehand': '\033[36m', # 青色
            'cache': '\033[33m',     # 黄色
        }
        
        color = colors.get(log_line.get('category', ''), '\033[0m')
        reset = '\033[0m'
        
        // 控制台彩色输出
        print(f"{color}[{log_line.get('category')}]{reset} {log_line.get('message')}")
        
        # 若有，记录执行时间
        if log_line.get('auxiliary', {}).get('execution_time'):
            exec_time = log_line['auxiliary']['execution_time']
            print(f"{exec_time['value']}{exec_time['unit']}")
        
        # 记录附加上下文
        auxiliary = log_line.get('auxiliary', {})
        if auxiliary and len(auxiliary) > 0:
            print('  Context:', json.dumps(auxiliary, indent=2))
        
        # 文件日志（可选）
        if self.log_file:
            log_entry = {
                'timestamp': timestamp,
                **log_line
            }
            with open(self.log_file, 'a') as f:
                f.write(json.dumps(log_entry) + '\n')

# 用法
logger = AdvancedLogger('./automation.log')
stagehand = Stagehand(
    env="BROWSERBASE",
    verbose=2,
    logger=logger.log
)
```

## 详细日志功能

### LLM 推理日志

启用对所有 LLM 交互的详细日志记录：

<CodeGroup>
```typescript TypeScript
const stagehand = new Stagehand({
  env: "BROWSERBASE",
  logInferenceToFile: true,  // 将创建 inference_summary/ 目录
  verbose: 2
});
```

```python Python
stagehand = Stagehand(
    env="BROWSERBASE",
    log_inference_to_file=True,  # 将创建 inference_summary/ 目录
    verbose=2
)
```
</CodeGroup>

## 最佳实践

<AccordionGroup>
<Accordion title="开发环境">
- 使用 `verbose: 2` 进行可视化调试
- 启用浏览器 DevTools 以检查元素
- 使用 `logInferenceToFile: true` 记录并捕获 LLM 决策
- 尽早实施结构化日志
</Accordion>

<Accordion title="生产环境">
- 使用 `verbose: 1` 在可见性与性能之间取得平衡
- 实施错误跟踪与告警
- 使用结构化 JSON 日志
- 监控会话成功率和执行时间
</Accordion>

<Accordion title="安全与合规">
- 切勿记录凭据或敏感数据
- 实施日志保留策略
- 保护日志文件和仪表盘
</Accordion>
</AccordionGroup>
