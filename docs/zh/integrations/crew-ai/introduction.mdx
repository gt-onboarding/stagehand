<div id="overview">
  ## 概览
</div>

本指南演示如何将 CrewAI 与 Browserbase 搭配使用，以创建能够自动化网页交互的智能代理。读完本指南后，你将了解如何：

- 使用 StagehandTool 配置 CrewAI
- 创建可与网站交互的代理
- 通过自然语言指令自动化浏览器任务
- 从网页提取结构化数据

<div id="when-youd-use-this">
  ## 适用场景
</div>

当你需要智能网页自动化时，CrewAI 集成非常适合：

- **研究自动化**：让代理在多个网站上检索与调研信息
- **数据采集**：从电商网站、招聘平台或新闻网站提取结构化数据
- **表单自动化**：依据特定条件自动填写并提交表单
- **多步工作流**：执行需要决策的复杂浏览器工作流

StagehandTool 封装了 Stagehand 的 Python SDK，使 CrewAI 代理能够控制真实的网页浏览器，并通过三个核心原语与网站交互：

1. **Act**：执行点击、输入或导航等操作
2. **Extract**：从网页中提取结构化数据
3. **Observe**：识别并分析页面元素
