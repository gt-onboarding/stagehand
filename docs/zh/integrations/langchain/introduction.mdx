<div id="overview">
  ## 概览
</div>

本指南演示如何将 Stagehand 与 LangChain JS 搭配使用，创建可自动化网页交互的智能代理。阅读完本指南后，你将了解如何：

- 在 LangChain JS 中配置 StagehandToolkit
- 创建能够浏览并与网站交互的代理
- 通过自然语言指令提取结构化数据
- 借助 LangGraph 构建复杂的自动化工作流

<div id="when-youd-use-this">
  ## 适用场景
</div>

当你需要具备高级推理能力的智能网页自动化时，LangChain JS 集成非常适合：

- **AI 驱动的研究**：创建代理，在多个网站上检索信息并综合整理结论
- **动态表单填写**：根据上下文需求自动填写复杂表单
- **数据提取工作流**：通过智能导航从多个来源提取并转换数据
- **多步网页流程**：执行需要决策与自适应的复杂浏览器工作流
