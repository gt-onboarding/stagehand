如果这是你**首次使用 Stagehand**，建议先试试 [Director](https://director.ai)。它是一个代理，能让你用自然语言搭建 Stagehand 工作流。你也可以通过我们的 [MCP 服务器](/zh/integrations/mcp/introduction) 体验 Stagehand。

否则，使用我们的 CLI 是上手 Stagehand 的最快方式。它会基于合理的默认配置与示例脚本，为你生成一个可直接运行的 Stagehand 应用。

<Note>
  本快速入门面向 **TypeScript**。如果需要 **Python**，请参见[安装指南](/zh/first-steps/installation)。
</Note>

## 1) 创建一个示例项目

<CodeGroup>
  ```bash Bash
  npx create-browser-app
  ```
</CodeGroup>

## 2) 运行

根据 CLI 提示进入项目目录并添加你的 API 密钥。然后运行示例脚本。

<CodeGroup>
  ```bash Bash
  cd my-stagehand-app # 进入项目目录
  cp .env.example .env  # 添加你的 API 密钥
  npm start # 运行示例脚本
  ```
</CodeGroup>

## 3) 使用 Stagehand（act、extract、observe）

脚手架包含一个 index.ts 文件，其中带有示例脚本。如下所示：

<CodeGroup>
  ```typescript TypeScript
  import "dotenv/config";
  import { Stagehand } from "@browserbasehq/stagehand";

  async function main() {
    const stagehand = new Stagehand({
      env: "BROWSERBASE"
    });

    await stagehand.init();

    console.log(`Stagehand Session Started`);
    console.log(`Watch live: https://browserbase.com/sessions/${stagehand.browserbaseSessionID}`);

    const page = stagehand.page;

    await page.goto("https://stagehand.dev");

    const extractResult = await page.extract("Extract the value proposition from the page.");
    console.log(`Extract result:\n`, extractResult);

    const actResult = await page.act("Click the 'Evals' button.");
    console.log(`Act result:\n`, actResult);

    const observeResult = await page.observe("What can I click on this page?");
    console.log(`Observe result:\n`, observeResult);

    const agent = await stagehand.agent({
      instructions: "You're a helpful assistant that can control a web browser.",
    });

    const agentResult = await agent.execute("What is the most accurate model to use in Stagehand?");
    console.log(`Agent result:\n`, agentResult);

    await stagehand.close();
  }

  main().catch((err) => {
    console.error(err);
    process.exit(1);
  });

  ```
</CodeGroup>

<Tip>
  使用前，请在 `.env` 中设置提供商密钥（例如 `OPENAI_API_KEY`）。如使用云浏览器，请同时添加 `BROWSERBASE_API_KEY` 与 `BROWSERBASE_PROJECT_ID`。
</Tip>
