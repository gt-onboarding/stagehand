title: Installation
description: 将 Stagehand 集成到现有项目中。
---

使用 TypeScript 或 Python 的 SDK 在当前应用中安装 Stagehand。

<Tip>
  对于 TypeScript/Node.js：我们强烈建议使用 Node.js 运行时来运行 Stagehand 脚本，而非 Deno 或 Bun 等较新的替代方案。

  由于不支持 [Playwright](https://github.com/search?q=repo:oven-sh/bun+playwright\&type=issues)，**Bun 不支持 Stagehand**。

  对于 Python：需要 Python 3.9+，并建议使用 [uv](https://docs.astral.sh/uv/) 管理虚拟环境。
</Tip>

<Tabs>
  <Tab title="TypeScript">
    ### 安装依赖 {#install-dependencies}

    <CodeGroup>
      ```bash npm
      npm install @browserbasehq/stagehand playwright zod
      ```

      ```bash pnpm
      pnpm add @browserbasehq/stagehand playwright zod
      ```

      ```bash yarn
      yarn add @browserbasehq/stagehand playwright zod
      ```
    </CodeGroup>

    <Tip>
      若计划本地运行，请先安装浏览器（一次即可）：`npx playwright install`。
      如使用云端浏览器会话，可跳过此步骤。
    </Tip>

    ### 配置环境 {#configure-environment}

    设置环境变量（或通过框架使用 `.env`）：

    <CodeGroup>
      ```bash Bash
      OPENAI_API_KEY=your_api_key
      BROWSERBASE_API_KEY=your_api_key
      BROWSERBASE_PROJECT_ID=your_project_id
      ```
    </CodeGroup>

    ### 在代码库中使用 {#use-in-your-codebase}

    在需要进行浏览器自动化的地方引入 Stagehand。

    <CodeGroup>
      ```typescript TypeScript
      import "dotenv/config";
      import { Stagehand } from "@browserbasehq/stagehand";
      import { z } from "zod";

      async function main() {
        const stagehand = new Stagehand({
          env: "BROWSERBASE"
        });

        await stagehand.init();
        const page = stagehand.page;

        await page.goto("https://example.com");
        
        // 在页面上执行操作
        await page.act("Click the sign in button");
        
        // 提取结构化数据
        const { title } = await page.extract({
          instruction: "extract the page title",
          schema: z.object({
            title: z.string(),
          }),
        });

        console.log(title);
        await stagehand.close();
      }

      main().catch((err) => {
        console.error(err);
        process.exit(1);
      });
      ```
    </CodeGroup>

    ### 配置环境 {#configure-environment}

    设置环境变量（或通过框架使用 `.env`）：

    <CodeGroup>
      ```bash Bash
      MODEL_API_KEY=your_api_key
      BROWSERBASE_API_KEY=your_api_key
      BROWSERBASE_PROJECT_ID=your_project_id
      ```
    </CodeGroup>

    ### 在代码库中使用 {#use-in-your-codebase}

    <CodeGroup>
      ```python Python
      import os
      import asyncio
      from stagehand import Stagehand

      async def main():
          stagehand = Stagehand(
              env="BROWSERBASE",
              model_api_key=os.getenv("MODEL_API_KEY")
          )
          await stagehand.init()
          page = stagehand.page
          
          await page.goto("https://example.com")
          
          # 在页面上执行操作
          await page.act("Click the sign in button")
          
          # 提取结构化数据
          result = await page.extract({
              "instruction": "extract the page title",
              "schema": {
                  "title": {
                      "type": "string"
                  }
              }
          })
          
          print(result["title"])
          await stagehand.close()

      if __name__ == "__main__":
          asyncio.run(main())
      ```
    </CodeGroup>
